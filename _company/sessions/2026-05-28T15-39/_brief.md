# 📋 작업 브리프

**원 명령:** 이번 세션은 PICTO LAB 실제 코드 구현 전용 세션이다.

금지:
- 일반 React App.jsx 구조 금지
- src/App.jsx 생성 금지
- src/utils/api.js 생성 금지
- ImageGenerator.jsx 생성 금지
- 외부 이미지 URL 사용 금지
- AI 이미지 생성기라는 이름 사용 금지
- 시뮬레이션 보고 금지
- 예시 코드만 제공 금지
- YouTube, PayPal, 결제, 관리자, 비즈니스 PoC 금지
- auto_planner.py 실행 금지

기준 문서:
- docs/picto-lab-direction.md

반드시 사용할 구조:
- src/app/page.tsx
- src/components/InputPanel.tsx
- src/components/ResultPanel.tsx
- src/lib/svg-renderer.ts
- src/lib/style-manager.ts
- src/lib/download.ts

서비스명:
- PICTO LAB

MVP 1단계 기능:
1. 키워드 입력
2. 용도 선택
3. 스타일 선택
4. 후보 4개 SVG 표시
5. SVG 다운로드
6. PNG 다운로드

키워드 예시:
- 나무
- 벤치
- 정원
- 도서관
- 강당
- 교실
- 휴식공간
- 동선

용도:
- 건축 도판
- 제안서/PPT
- 공공기관 보고서
- 브랜드/서비스
- 앱/웹 UI

스타일:
- Line
- Solid
- Bold Line
- Minimal

구현 조건:
- Next.js App Router 기준으로 구현
- TypeScript 사용
- 외부 API 없이 템플릿/파라메트릭 SVG 문자열로 후보 4개 생성
- 후보 4개는 실제 SVG로 렌더링
- SVG 다운로드는 Blob으로 구현
- PNG 다운로드는 SVG를 canvas에 렌더링 후 PNG로 저장
- 구현 후 npm run build를 실제 실행

지금 바로 실제 파일을 수정해.
설명이나 계획을 먼저 쓰지 말고, 파일 수정 후 아래 형식으로만 보고해.
