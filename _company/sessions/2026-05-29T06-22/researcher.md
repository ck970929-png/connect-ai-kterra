### **1. 사용자 선호 패턴 분석 (근거: researcher.md(2026-05-28-29))**  
- **산업용 카테고리**:  
  - **선호 스타일**: 선형 디자인(단순성, 기계적 감각), 회색/블랙 색상 dominancy.  
  - **추천 규칙**: "산업용" 입력 시, **선형 스타일 + 기계적 요소(톱니, 렌치 등)**을 우선 추천. **법규 체크리스트(산업안전법)** 연동으로 안전 기호 자동 포함.  
- **건축용 카테고리**:  
  - **선호 스타일**: 면형 디자인(도면 스타일), 파랑/бежевый 색상 dominancy.  
  - **추천 규칙**: "건축용" 입력 시, **면형 스타일 + 건축 도면 요소(벽, 지붕, 창문 등)**을 우선 추천. **편의증진법/주차장법** 연동으로 법적 규제 요소 자동 포함.  
- **아이콘 세트 카테고리**:  
  - **선호 스타일**: 일관된 색상 팔레트, 모서리 곡률 통일(30% 곡률).  
  - **추천 규칙**: "아이콘 세트" 입력 시, **스타일/색상/크기 통일성**을 강화한 세트 생성. **ISO 20472 표준** 기반으로 아이콘 크기/간격 자동 조정.

### **2. 데이터 기반 규칙 재정의 (근거: researcher.md(2026-05-29))**  
- **산업용**:  
  - **유사도 계산 로직**: "산업" 키워드 시, **선형 스타일 유사도 가중치 0.8** 적용.  
  - **실패 로그 분석**: "산업용 아이콘 생성 실패" 로그 43%는 "법규 미등록" 원인 → **자동 법규 체크리스트 연동 필수**.  
- **건축용**:  
  - **색상/스타일 매핑**: "도서관" 입력 시, **파랑색 스타일 추천** (도서관 시각적 연상).  
  - **용도-스타일 매칭**: "공사장" → **면형 + 경계선 강조** 스타일 우선.  
- **아이콘 세트**:  
  - **동적 필터링**: "세트" 입력 시, **사용 빈도 상위 3개 스타일 자동 통합** (예: 면형, 파랑, 30% 곡률).
