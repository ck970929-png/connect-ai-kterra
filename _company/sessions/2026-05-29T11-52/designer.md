# 🎨 츄 — PICTO LAB의 스타일 추천 시스템을 사용자 선호 데이터와 연동할 수 있는 UI/UX 흐름으로 정리하세요. 선형/면형/굵은 선형 스타일 선택 기준과 데이터 반영 방식을 포함하세요.

**PICTO LAB 스타일 추천 시스템 UI/UX 흐름 정리**  
1. **사용자 선호 데이터 연동 흐름**  
   - **단어 입력 → AI 검색/추천** 단계에서, 사용자 입력 단어에 대한 의미 분석 후, **스타일 선택 섹션**에 **동적 스타일 제안 UI**를 추가합니다.  
     - 예: "경고" 단어 입력 시, AI가 "면형" 스타일을 추천하고, 사용자가 "굵은 선형"으로 선택하면, **색상/선두께/모서리 곡률**이 사용자 선호 데이터에 따라 자동 조정됩니다.  
     - [근거: designer.md, PICTO_LAB_FINAL_SPEC.md]  

   - **스타일 선택 섹션**은 좌측에 **선형/면형/굵은 선형** 옵션을 시각적으로 구분한 버튼 형태로 배치하고, 오른쪽에 **결과 후보 미리보기** 영역을 둡니다.  
     - **도면 레이아웃**은 "용도-스타일 매핑" 표시(예: "성공" → 굵은 선형)을 통해 사용자 혼란 최소화.  
     - [근거: researcher.md, designer.md]  

2. **선형/면형/굵은 선형 선택 기준**  
   - **선형**: 단순하고 직관적인 표현, 건축 도면에 적합.  
   - **면형**: 색상 강조, 시각적 주목도를 높이기 위한 선택.  
   - **굵은 선형**: 경계 강조, 경고/특수 기호 표현에 적합.  
   - **데이터 반영 방식**: 사용자 선택 이력과 선호 색상/선두께를 기반으로, AI가 **스타일 추천 알고리즘**을 통해 동적으로 조정.  
     - 예: 사용자가 "면형"을 선호하면, 이후 "성공" 단어 입력 시 기본 스타일로 "면형" 제안.  
     - [근거: designer.md, HANDOVER_KTERRA.md]  

3. **데이터 연동 UI 요소**  
   - **툴팁**: "색상은 사용자 선호 데이터 기반으로 동적으로 변환됩니다" (동적 색상 변환 시 표시).  
   - **사용자 프로필 연동**: 사용자가 "색상/선두께"를 직접 설정할 수 있는 **설정 섹션** 추가.  
   - [근거: designer.md, PICTO_LAB_FINAL_SPEC.md]
