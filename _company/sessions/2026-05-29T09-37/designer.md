## 🔧 1. 스타일 시스템 정의  
### ✅ 선형/면형/굵은 선형 표현 규칙  
- **선형 (Linear):**  
  - **선 두께:** 1.5px (기본)  
  - **모서리:** 둥근 곡률 (R=2px)  
  - **적용 예:** "도면", "기초" 등 구조적 요소  
  - **배치:** 좌측에 위치 (근거: designer.md)  
- **면형 (Surface):**  
  - **선 두께:** 3px  
  - **모서리:** 둥근 곡률 (R=4px)  
  - **색상:** `#F4A261` 강조 색상 사용 (근거: designer.md)  
  - **적용 예:** "경고", "성공" 등 시각적 강조 필요 시  
- **굵은 선형 (Thick Stroke):**  
  - **선 두께:** 4.5px  
  - **모서리:** 직선 처리 (R=0)  
  - **적용 예:** "스마트 팩토리", "AI 관제탑" 등 Kterra 전용 아이콘 (근거: designer.md)

## 🎨 2. 색상 팔레트 및 선명도 기준  
### 🎨 색상 팔레트  
- **기본 색상:**  
  - `#2A363B` (深夜 블랙) — 배경/텍스트  
  - `#F4A261` (연한 주황) — 강조 색상 (사용자 선호 데이터 기반 동적 변환)  
  - `#2A363B` → `#F4A261` 전환 시 **토oltip:** "색상은 사용자 선호 데이터 기반으로 동적으로 변환됩니다" (근거: designer.md)  
- **보조 색상:**  
  - `#4A5568` (그레이) — 경계선/분할선  
  - `#E2E8F0` (아이보리) — 비어 있는 상태(empty state) 배경  

### 🔦 선명도 기준  
- **색상 대비:**  
  - 주 색상(`#F4A261`)과 배경(`#2A363B`) 간 대비 비율 **12:1** 이상 유지 (WCAG 2.1 AA 기준)  
- **선명도 조절:**  
  - 사용자 선호 데이터 기반으로 **선명도 70-90%** 범위 조절 (근거: researcher.md)

## 📐 3. 도판 표현 규칙  
### 📐 용도-스타일 매핑 표시  
| **용도**      | **추천 스타일**   | **색상**       | **도판 톤앤매너**              |  
|---------------|-------------------|----------------|-------------------------------|  
| 경고          | 면형              | `#F4A261`      | 경계선 두께 2px, 텍스트 왼쪽 정렬 |  
| 성공          | 굵은 선형         | `#F4A261`      | 텍스트 중앙 정렬, 배경 흐림 효과 |  
| 구조적 요소   | 선형              | `#2A363B`      | 텍스트 오른쪽 정렬, 단순 선 표현 |  

### 📐 도면 레이아웃  
- **도면 배치:**  
  - **용도-스타일 매핑** 표시 (예: "경고" → 면형, "성공" → 굵은 선형) (근거: researcher.md)  
- **레이아웃 패턴:**  
  - **좌측:** 스타일 선택 영역 (선형/면형/굵은 선형)  
  - **우측:** 결과 후보 및 다운로드 버튼 (근거: designer.md)

## 🔄 4. 사용자 선호 데이터 반영 방식  
### 🔄 동적 스타일 추천 시스템  
1. **데이터 수집:**  
   - 사용자 입력 단어 기반으로 **최소 단어 데이터셋 20개** 분석 (근거: designer.md)  
   - 선호 스타일, 색상, 선명도 기록  
2. **추천 로직:**  
   - **AI 검색/추천** → **스타일 선택** → **SVG/PNG 다운로드** 흐름 (근거: PICTO_LAB_FINAL_SPEC.md)  
   - 사용자 히스토리 기반으로 **선형/면형/굵은 선형** 시각적 구분 추가 (근거: designer.md)  
3. **피드백 메커니즘:**  
   - **마이크로 인터랙션:** 스타일 선택 시 **Glassmorphism 패널 블러 효과** 적용 (여백 10px, 투명도 30%) (근거: designer.md)
