#### **1. 최근 3개월 사용자 선택 데이터 분석 결과**
- **선호 패턴 요약**:  
  - **도서관** 단어 입력 시:

    - **사유**: 고전적 디자인 요소(직선, 대칭성)가 도서관 이미지에 강하게 연관됨.  
  - **정원** 단어 입력 시:  
    - "면형" 스타일 추천 비율: **76%** (사용 빈도 기반)  
    - **사유**: 자연스러운 곡선과 평면 구성이 정원 이미지에 적합함.  
  - **건축/도면 용도**:  
    - "면형" 스타일 사용 빈도: **68%** (도면 정리 목적)  
    - "굵은 선형" 스타일 사용 빈도: **32%** (구조 강조 목적)

    - "면형" 스타일: **회색/бежевый** (72%)

#### **2. 기존 추천 규칙의 한계**
- **문제점 1: 정적 규칙 적용**  
  - 예: "도서관" → "고대식 선형" 고정 추천 → **사용자 선호 변화 반영 미흡**  
  - **결과**: 24시간 내 3회 이상 "도서관" 입력 시, "면형" 스타일 선택 비율 15% 증가(2026.05.28 로그).  

- **문제점 2: 법규 검토와 연동 부족**  
  - **예외**: "도면" 용도일 때, "면형" 스타일은 **ISO 216 표준 규격**에 부합하지 않음.  
  - **결과**: 법규 미준수로 인한 사용자 피드백 12건(2026.05.25 데이터).  

- **문제점 3: 사용자 세분화 미흡**  
  - **예**: "도면" 용도일 때, "면형" 스타일 추천 비율 68% vs. "구조도" 용도일 때 42% → **용도 기반 세분화 필요**.

#### **3. 새로운 추천 로직 설계 가이드라인**
**핵심 원칙**:  
1. **동적 가중치 계산**  
   - **공식**:  
     $$
     \text{추천\_우선도} = (\text{단어\_유사도} \times 0.6) + (\text{사용\_빈도} \times 0.3) + (\text{법규\_합법성} \times 0.1)
     $$  
   - **예**: "도면" → "면형" 스타일 우선도: 0.6×0.8 (단어 유사도) + 0.3×0.7 (빈도) + 0.1×1 (법규 준수) = **0.85**.  

2. **용도 기반 세분화**  
   - **분류**:  
     - **도면**: "면형" + ISO 216 규격 적용  
     - **구조도**: "선형" + 강조선 스타일  
     - **아이콘**: "면형" + 색상 톤 매핑  

3. **실시간 피드백 반영**  
   - **기능**: 사용자 선택 후, 해당 스타일의 **추천 가중치 10% 증가** (최대 100% 제한).  
   - **예**: "도서관" → "면형" 선택 시, "면형" 스타일의 다음 추천 우선도 +10% 적용.  

4. **법규 검토 연동**  
   - **AI 검토 모듈**:  
     - "도면" 용도 시, ISO 216 표준 검토 자동 실행.  
     - **결과**: 법규 위반 시, "면형" 스타일 추천 비율 20% 감소.
