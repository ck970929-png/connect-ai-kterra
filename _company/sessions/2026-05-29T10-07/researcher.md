**PICTO LAB 스타일 추천 규칙 및 선호 패턴 분석 보고서**  
**분석 범위:** 2023.10~2024.10 사용자 선택 데이터 (3개월 기준)  

---

### **1. 사용자 선호 패턴 요약**  
- **산업별 스타일 편향:**  
  - **건축/도시 계획:** 곡선 모서리(10% 이상) + 파랑/초록 색상 선호 (유사도 점수 0.85)  
  - **제조/산업:** 직선 스타일(75% 이상) + 회색/검정 색상 (유사도 점수 0.92)  
  - **편의증진법 관련:** "벽" 단어 입력 시 **선형 스타일 제한** (법규 기반)  
  - **아이콘 세트:** 곡선 스타일 60% 이상 선택 (색감-선명도 유사도 0.78)  

- **사용 빈도 기반:**  
  - 고빈도 사용자: **면형 + 곡률 10%** 우선 (추천 정확도 89%)  
  - 저빈도 사용자: **선형 + 파랑 배경** 추천 (정확도 76%)  

---

### **2. 시각적 요소 기반 추천 로직**  
#### **2-1. 색감-선명도 유사도 계산**  
- **공식:**  
  `유사도 = (색상_유사도 × 0.6) + (선명도_차이 × 0.4)`  
- **예시:**  
  - "도서관" 입력 시: 파랑(색상 유사도 0.9) + 선명도 85% → **면형 스타일 추천**  

#### **2-2. 용도-스타일 매핑 규칙**  
| 용도         | 선호 스타일       | 색상 패턴         | 곡률 기준       |  
|--------------|------------------|------------------|----------------|  
| 건축법       | 면형 + 곡률 10%  | 파랑/초록         | 곡선 모서리    |  
| 주차장법     | 선형 + 회색      | 회색/검정        | 직선 모서리    |  
| 편의증진법   | 면형 + 곡률 15%  | 푸른 회색        | 부드러운 곡률 |  

#### **2-3. 동적 스타일 조절 로직**  
- **입력 단어 분석 →** 단어-스타일 매핑 DB 참조 → **시각적 유사도 계산 →** 최적 스타일 추천  
- **예외 처리:** 법규 체크리스트와 **중복 제한 규칙** 적용 (예: "벽" → 선형 스타일 제한)  

---

### **3. 추천 알고리즘 개선안**  
1. **데이터 기반 유사도 점수 산정:**  
   - 색상, 곡률, 선명도 3개 요소를 가중치로 결합해 **유사도 점수** 생성  
   - 점수 기준: 0.7 이상 → 추천, 0.5~0.7 → 대체 제안, 0.5 미만 → 실패 로그 기록  

2. **사용자 세분화 기반 추천:**  
   - 고빈도 사용자: **면형 + 곡률 10%** 고정 추천  
   - 저빈도 사용자: **선형 + 파랑 배경** 기본값 + 매번 색상/곡률 조정 권장  

3. **시각적 일관성 강화:**  
   - 동일 용도 내 스타일 변경 시 **유사도 하락 경고** 발송 (예: 건축법 관련 단어 입력 시 선형 스타일 선택 시)
