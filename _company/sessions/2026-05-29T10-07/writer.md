### 📄 **API 문서: 픽토그램 스타일 추천 엔드포인트**  
**URL**: `/api/v1/picto/recommend`  
**Method**: `POST`  

#### 📦 요청 바디 (JSON)  
```json
{
  "input_word": "도서관", // 사용자 입력 단어
  "preferred_style": "면형", // 선호 스타일 (선형/면형/굵은 선형)
  "brightness_range": [70, 90], // 선명도 범위 (WCAG 2.1 AA 준수)
  "exclude_terms": ["벽"] // 법규 제외 키워드
}
```  

#### 📌 응답 예시  
```json
{
  "recommended_style": "면형",
  "color_palette": ["#007BFF", "#00BFFF"],
  "brightness": 85,
  "explanation": "도서관 단어 입력 시 파랑 계열 색상과 면형 스타일이 유사도 0.85로 추천됨 (법규 제외 키워드 '벽' 적용)"
}
```

### 📖 **사용자 가이드: 시각 요소 기반 추천 과정**  
1. **단어 분석**: 입력 단어를 기반으로 DB에서 스타일 매핑 규칙 검색  
   - 예: "도서관" → 파랑/면형 선호 (유사도 0.85)  
2. **색상·선명도 계산**:  
   - 공식: `유사도 = (색상_유사도 × 0.6) + (선명도_차이 × 0.4)`  
   - 결과: 선명도 85% 범위 내 파랑 계열 색상 추천  
3. **법규 검토**:  
   - "벽" 입력 시 선형 스타일 제외 (법규 제한 적용)  
4. **동적 조정**:  
   - 고빈도 사용자: 면형 + 곡률 10% 고정 추천  
   - 저빈도 사용자: 선형 + 파랑 배경 기본값 권장
