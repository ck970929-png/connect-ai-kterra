# 🔍 카리나 — PICTO LAB 사용자 선호 데이터를 기반으로 스타일 추천 알고리즘을 최적화해 주세요. 법규 검토와 병행해 자동화 규칙을 확장해야 합니다.

**작업 결정:**  
PICTO LAB의 스타일 추천 알고리즘을 법규 검토와 연동한 **동적 규칙 엔진**으로 최적화하고, 사용자 선호 데이터 기반의 3단계 추천 로직을 재설계합니다.  

**실행 계획:**  
1. **법규-스타일 제한 규칙 통합**  
   - 건축법/주차장법 체크리스트를 기반으로 "벽" 단어에 대한 선형 스타일 제한 등 규칙을 자동화 (예: `단어 == "벽" → 스타일 제한 = ["선형"]`).  
   - 법제처 API로 실시간 법령 검증 절차를 추가 (출처: `PICTO_LAB_FINAL_SPEC.md` 참고).  

2. **3단계 추천 로직 재정의**  
   - **단계 1:** 사용자 세분화 (고빈도/저빈도) → 고빈도 사용자에게 "면형 + 곡률 10%" 우선 (메모리: `고빈도 사용자 → 면형 + 곡률 10%`).  
   - **단계 2:** 용도별 스타일 매핑 (산업용=선형, 건축용=면형) → 최근 3개월 데이터 기반 (출처: `PICTO_LAB_FINAL_SPEC.md`).  
   - **단계 3:** 시각적 일관성 검증 (색감-선명도 연계) → 예외 사례 기록 (출처: `Researcher 개인 메모리`).  

**책임자 분배:**  
- **Researcher** (데이터 분석 및 규칙 설계)  
- **Developer** (알고리즘 구현 및 법제처 API 연동)  

**자가검증:** 사실 3개 / 추측 0개
