### 🎨 **브랜드 시각 언어 (Brand Visual Language)**  
1. **컬러 시스템**  
   - 주 색상: `#2A5CAA` (스마트 건축 감성)  
   - 보조 색상: `#F59E0B` (창의성), `#4F46E5` (신뢰)  
   - 제약: 건축 도판 표현 시 `#FFFFFF` 흰색 배경 강제 적용  

2. **타이포그래피**  
   - 제목: **Inter** (14pt, 굵은 선)  
   - 본문: **Nunito Sans** (12pt, 중간 굵기)  
   - 경고/오류: **Montserrat** (10pt, 경량)  

3. **로고 시스템**  
   - 로고: 직사각형 프레임 + 선형/면형 픽토그램 조합 (예: 도서관 → 빌딩 라인 + 책 페이지 라인)  
   - 확장: `Picto Lab` 텍스트는 `#4F46E5` 색상, 픽토그램은 `#2A5CAA`로 구분  

---

### 🖼️ **픽토그램 스타일 시스템**  
| 스타일 | 선 두께 | 모서리 | 단순화 비율 | 적용 예시 |  
|--------|--------|--------|------------|-----------|  
| **선형** | 2px | 둥글음 | 80% | 건축, 도서관 |  
| **면형** | 4px | 직선 | 60% | 정원, 벤치 |  
| **굵은 선형** | 6px | 둥글음 | 90% | 경계, 경관 |  

- **동적 추천 로직**:  
  - 사용자 활동 빈도 3회 이상 → '면형' 우선 (85% 우선순위)  
  - 도서관 검색 5회 이상 → '고대식 선형' 강제 적용 (우선순위 95%)  

---

### 🏗️ **건축 도판 표현 방식**  
1. **도판 레이아웃**  
   - 상단: 브랜드 로고 + 키워드 입력 필드 (120px 높이)  
   - 중앙: 4개 결과 후보 (2x2 그리드, 300x300px 크기)  
   - 하단: 다운로드 버튼 (선형/면형/굵은 선형)  

2. **도판 색상 규칙**  
   - 기본 배경: `#FFFFFF`  
   - 픽토그램: `#2A5CAA`  
   - 하이라이트: `#F59E0B` (선택된 아이콘에만 적용)  

---

### 🖥️ **웹 UI 톤 앤 매너**  
1. **인터랙션 피드백**  
   - 입력 필드: `#F59E0B` 테두리, 포커스 시 `#4F46E5` 색상 변환  
   - 결과 후보: 마우스 오버 시 `#E0E7FF` 텍스처 효과 추가  

2. **에러/경고 처리**  
   - 키워드 입력 시:  
     - "도서관" → 정확한 픽토그램 표시  
     - "식당" → "도서관"과 유사한 라인 스타일 경고 메시지 표시  

3. **접근성**  
   - 색상 대비: `#2A5CAA` vs `#FFFFFF` (WCAG 2.1 AA 준수)  
   - 키보드 네비게이션: 결과 후보 클릭 시 `#4F46E5` 하이라이트
