#### 1. **추천 시스템 구조 재정의**  
- **데이터 기반 스타일 권장 규칙**:  
  - 산업/건축 단어 입력 시, **면형 스타일 권장률 68%**, **선형 스타일 32%**로 규칙화 (예: "도어" → 면형, "주차 공간" → 선형).  
  - 사용자 선호 데이터베이스를 기반으로 **실시간 스타일 추천 알고리즘** 구현 (예: 최근 3개월 데이터 기반 유사도 계산).  
  - 법규 준수 검증 모듈: **법제처 API 연동**으로 건축법, 주차법 등 규제와의 일치 여부 자동 검증 (불일치 시 대체 스타일 제안).  

- **AI 추천 엔진 구조**:  
  - **단계 1**: 키워드 → AI 검색 (법규/용도 매핑).  
  - **단계 2**: 스타일 선택 → 사용자 선호 DB + 산업 규칙 기반 권장.  
  - **단계 3**: SVG/PNG 다운로드 → 크레딧 소모/유료 기능 유도.  

---

#### 2. **유료 기능 구분 및 크레딧 시스템 통합**  
- **유료 기능 분류**:  
  1. **기본 기능 (무료)**:  
     - 일반 스타일 추천 (법규 검증 제외).  
     - 기본 해상도 SVG/PNG 다운로드.  
  2. **프리미엄 기능 (유료)**:  
     - **법규 준수 검증 포함 스타일 추천** (법제처 API 연동).  
     - 고해상도 PNG (300dpi 이상) 다운로드.  
     - 스타일 커스터마이징 (모서리 곡률, 색상 조정).  
     - 무제한 크레딧 패키지 (월별/분기별 구독).  

- **크레딧 시스템 구조**:  
  - **크레딧 유형**:  
    - **기본 크레딧**: 일반 스타일 생성 (1개 = 1크레딧).  
    - **프리미엄 크레딧**: 법규 검증 + 고해상도 다운로드 (1개 = 2크레딧).  
  - **결제 모델**:  
    - **구독 모델**: 월 5만원 → 500 기본 크레딧 + 100 프리미엄 크레딧.  
    - **유료 패키지**: 10만원 → 1,000 기본 크레딧 + 200 프리미엄 크레딧.  
    - **프리미엄 구독**: 20만원 → 무제한 크레딧 + 1:1 고객 지원.  

- **크레딧 사용 흐름**:  
  1. 사용자가 프리미엄 기능 선택 시, **크레딧 소모 알림** 표시.  
  2. 잔여 크레딧이 0이면, **유료 구독/패키지 구매 유도 팝업** 노출.
