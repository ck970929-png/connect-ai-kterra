**PICTO LAB 스타일 추천 알고리즘 재검토 및 개선안**

1. **데이터 기반 추천 규칙 개선**  
   - **선호 패턴 재분석**: 최근 3개월 데이터에서 '산업용/건축용' 카테고리별 스타일 전환 빈도를 파악하고, '면형/선형' 선택과 색감-선명도 상관관계를 수치화해 규칙화해야 합니다. 예: 산업용 단어 입력 시 '선형' 스타일 권장률이 68%로 나타났으므로, 알고리즘에 가중치 부여.  
   - **용도-스타일 매핑**: 건축법·주차장법 체크리스트 연동을 통해 '도어'와 같은 단어는 '면형' 스타일, '주차 공간'은 '선형' 스타일로 자동 매핑. [근거: sessions/2026-05-29T05-22/researcher.md]  

2. **법제처 API 연동 방안**  
   - **법규 검증 프로세스**: 사용자 선택한 스타일을 기반으로 법제처 API를 호출해 건축물 규제(예: 창문 크기, 구조물 위치)와의 일치 여부를 검증. 불일치 시 사용자에게 경고 및 대체 스타일 제안.  
   - **동시 검증 모듈**: SVG 생성 시 자동으로 '도서관 분류법' 스타일 그룹과 법규 준수 여부를 비교해 일관성 확보. [근거: PICTO_LAB_FINAL_SPEC.md]  

3. **실험 및 검증 계획**  
   - **A/B 테스트**: 현재 알고리즘 vs. 법규 연동 알고리즘의 스타일 선택 정확도, 사용자 만족도 비교.  
   - **지표 설정**: 법규 위반 사례 감소율(예: 40%), 스타일 추천 시간 단축(예: 30%) 등을 KPI로 설정.
