### 🎨 브랜드 색상 시스템  
- **Primary (브랜드 컬러):** `#2A3B5A` (딥 블루, 100%) → `#F5F7FA` (아이보리, 10%)  
  - **적용 영역:** 로고, CTA 버튼, 헤더 배경  
- **Secondary (보조 컬러):** `#FF6B6B` (레드) / `#6C63FF` (보라)  
  - **적용 영역:** 스타일 선택 메뉴, 알림/오류 UI, 강조 요소  
- **Neutral (네이트럴):** `#D3D3D3` (그래이) / `#FFFFFF` (화이트)  
  - **적용 영역:** 배경, 텍스트, 아이콘 테두리  

---

### 🖋️ 타이포그래피  
- **Sans-serif:** `Montserrat` (폰트 가중도: 400, 600, 700)  
  - **적용 영역:** 제목, 버튼, 메뉴 항목  
- **Serif:** `Noto Serif` (폰트 가중도: 400, 600)  
  - **적용 영역:** 서술문, 설명 텍스트, 라벨  
- **텍스트 스타일:**  
  - **표준:** 16px / 1.5x 라인 높이  
  - **제목:** 24px / 1.8x 라인 높이 (헤더), 18px / 1.5x 라인 높이 (세부 항목)  

---

### 🎯 아이콘 스타일  
- **스타일:** 라인 아이콘 (선형, 1.5px 스톱)  
  - **색상:** `#2A3B5A` (기본), `#FF6B6B` (강조), `#6C63FF` (보조)  
  - **변형:**  
    - **선형:** 단순한 선으로 구성 (예: "건축" 아이콘: 직사각형 + 외곽선)  
    - **면형:** 평면 디자인 (예: "도서관" 아이콘: 책 형태 + 그림자 효과)  
    - ** 굵은 선형:** 3px 스토퍼 사용 (예: "경고" 아이콘: 삼각형 + 둥근 모서리)  

---

### 📐 레이아웃 패턴  
- **그리드 시스템:** 12px 기준 그리드 (모든 컨텐츠 정렬)  
- **간격:**  
  - **내부 간격:** 16px (버튼, 요소 간 간격)  
  - **외부 간격:** 24px (섹션 간 빈 공간)  
- **레이아웃 구조:**  
  - **헤더:** 로고(16px) + 메뉴(12px 간격) + 스타일 선택 메뉴(1.5x 높이)  
  - **컨텐츠 영역:** 단어 입력창(24px 높이) + 결과 후보(3x 그리드)  
  - **푸터:** 브랜드 로고 + 저작권 텍스트(14px)  

---

### 🌀 마이크로 인터랙션  
- **호버 효과:**  
  - **버튼:** 2px 둥근 모서리 + `#FF6B6B` 색상 강조  
  - **아이콘:** 1.2x 크기 확대 + 50ms 애니메이션  
- **포커스 상태:**  
  - **입력창:** 1px 테두리 + `#6C63FF` 색상 강조  
  - **메뉴 항목:** 100ms 블링 효과 (색상: `#F5F7FA`)  
- **로딩 상태:**  
  - **SVG 생성 중:** 회전하는 원형 로딩 아이콘 (16px, 200ms 주기)  
  - **다운로드 버튼:** 200ms 투명도 감소 효과 (0.8 → 0.5)
