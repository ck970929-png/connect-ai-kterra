### 🎯 **UX 카피 스타일 가이드**  
**감성 톤:** "직觉적인 창의성, 마법 같은 단순함"  
**핵심 문구 패턴:**  
1. **입력창 프롬프트**  
   - "어떤 아이콘을 원하시나요? 단어 하나로, 창의적인 마법을 시작해보세요."  
   - "예: '도서관', '경고', '자연' — 단어를 입력하면, 당신의 아이콘을 만들어 드립니다."  

2. **스타일 선택 메뉴 라벨**  
   - "스타일을 선택해주세요"  
   - "선형: 정제된 형태 / 면형: 평면 디자인 / 굵은 선형: 강렬한 시각적 인상"  

3. **결과 후보 설명**  
   - "이 아이콘들이 당신의 이야기를 완성할 수 있습니다"  
   - "하나를 선택하면, SVG/PNG로 다운로드할 수 있어요"  

4. **다운로드 버튼 CTA**  
   - "마법을 완성하려면, 이 아이콘을 선택해주세요"  
   - "SVG로 내려받기 / PNG로 저장하기"

### 📄 **도판 문구 예시**  
**제목:**  
"단어 하나로, 당신의 아이콘을 만드는 마법"  

**본문:**  
"PICTO LAB은 단순한 단어를 입력할 때마다, 일관된 픽토그램을 생성하는 서비스입니다.  
선형, 면형, 굵은 선형 스타일로 당신의 아이콘을 완성해보세요.  
SVG/PNG로 즉시 다운로드하여, 프로젝트에 활용해보세요."  

**CTA:**  
"시작하기" / "예시 보기"

### 📊 **발표문 템플릿 (슬라이드 기준)**  
**슬라이드 1: 제목**  
"단어 하나로, 당신의 아이콘을 만드는 마법"  
- **브랜드 로고 + 타이포그래피**  
- **색상:** `#2A3B5A` (딥 블루) + `#FF6B6B` (레드)  

**슬라이드 2: 시각 언어 시스템**  
- **브랜드 색상:** "딥 블루 + 레드, 보라, 화이트"  
- **타이포그래피:** "Montserrat(400, 600), Noto Serif(400)"  
- **아이콘 스타일:** "선형, 면형, 굵은 선형"  

**슬라이드 3: 사용자 경험 예시**  
- **입력창:** "도서관" → "면형 스타일로 생성"  
- **결과:** "아이콘 후보 3개 + SVG/PNG 다운로드"  
- **CTA:** "다운로드" / "다른 스타일 시도"  

**슬라이드 4: 마이크로 인터랙션**  
- **호버 효과:** "버튼: 레드 강조 + 2px 둥근 모서리"  
- **포커스 상태:** "입력창: 보라 테두리 + 1px 강조"  

**슬라이드 5: 결론**  
"단어를 입력하면, 당신의 아이콘을 만드는 마법.  
PICTO LAB은 당신의 창의성을 완성하는 픽토그램 생성 서비스입니다."
