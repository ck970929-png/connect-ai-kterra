### 📈 PICTO LAB 수익화 전략 (무료/유료 구분 + 팩 판매 + 구독 모델 + 광고 수익)

#### ✅ 무료 기능 (Free Tier)
- **기본 기능**: 단어 입력 → 4가지 후보 추천 (선형/면형/굵은 선형)
- **제한**: 5회/일, PNG/SVG 다운로드 제한 1회
- **추가 기능**: 사용자 피드백 수집, AI 추천 알고리즘 학습

#### 💰 유료 기능 (Paid Tier)
- **기본 기능**: 무제한 생성, 고급 스타일 (3D/모션 효과)
- **추가 기능**: 
  - PNG/SVG 고해상도 다운로드 (300dpi)
  - 팀 사용권 (5명 내외)
  - 맞춤형 스타일 템플릿 생성
  - API 연동 기능 (개발자용)
- **가격**: 월 $9.99 / 연 $99.99 (15% 할인)

#### 🎁 팩 판매 (Bundled Sales)
1. **학생 패키지**: $4.99 (월 3회 생성, 기본 스타일 2종)
2. **프로 패키지**: $19.99 (월 10회 생성, 고급 스타일 5종 + API 연동)
3. **기업 협상 패키지**: 맞춤형 API 연동 + 10명 이상 팀 사용권

#### 🔄 구독 모델 (Subscription Model)
- **월간 구독**: $8.99 (기본 기능 + 10회 생성)
- **연간 구독**: $89.99 (월간 20% 할인 + 추가 스타일 3종)
- **구독 혜택**: 
  - 매월 새로운 스타일 템플릿 제공
  - 사용자 데이터 통계 제공
  - 우선 개발자 지원

#### 📢 광고 수익 (Ad Revenue)
- **제한적 광고**: 
  - 사용자 피드백 수집 시 관련 광고 표시 (1회/일)
  - AI 추천 결과 하단에 광고 배너 (30% 브랜드 참여율)
- **수익 비율**: 광고 수익의 50%를 수익화 (유료 사용자에게만 노출)
