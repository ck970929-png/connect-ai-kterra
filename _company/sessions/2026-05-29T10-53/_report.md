### **2. UI/UX 흐름 및 시각 언어 업그레이드**  
#### **핵심 흐름**  
1. **단어 입력 → 스타일 선택 → 결과 후보 → 다운로드**  
2. **강점**: 간결한 인터페이스, 직관적인 아이콘 선택, 빠른 결과 제공  
3. **개선 사항**:  
   - 사용자 피드백 수집 시스템 추가  
   - 색상/선명도 동적 조정 기능 강화  

#### **브랜드 시각 언어**  
- **색상 시스템**:  
  - **주 색상**: `#F4A261` (에너지감 있는 오렌지)  
  - **보조 색상**: `#2A363B` (저조도 검정)  
  - **배경색**: `#FFFFFF` (흰색)  
- **폰트**: `Inter` (미니멀, 고해상도 지원)  
- **타이포그래피**:  
  - 제목: `32px`, 평문: `16px` (간격 1.5x)  
  - 라인 높이: `1.8` (편안한 읽기 경험)  

#### **도판 표현**  
- **상단**: 단어 입력 필드 + 사용자 피드백 수집 버튼 ("이 스타일이 마음에 드나요?")  
- **중간**: 스타일 선택 영역 (선형/면형/굵은 선형) + 동적 색상/선명도 표시  
- **하단**: 결과 후보 (4개) + 다운로드 버튼 (PNG/SVG)  
- **도판 톤앤매너**:  
  - "픽토그램은 사용자 선호 데이터와 용도 기반으로 생성됩니다."  
  - "색상은 사용자 선호 데이터 기반으로 동적으로 변환됩니다."

### **3. 기술 구현 방안**  
#### **파일 관리 구조**  
- **/assets**:  
  - `/fonts` (Inter, Lato 등)  
  - `/colors` (주 색상 `#F4A261`, 보조 색상 `#2A363B`)  
  - `/icons` (선형/면형/굵은 선형 아이콘 세트)  
- **/src**:  
  - `/components` (UI 컴포넌트: 스타일 선택 영역, 결과 후보, 다운로드 버튼)  
  - `/services` (색상/선명도 동적 조정 로직, 사용자 피드백 수집 API)  
  - `/utils` (파일 다운로드 유틸, 색상 변환 함수)  
  - `/config` (환경 설정: 색상 테마, API 키, 로그인 설정)  
- **/docs**:  
  - `style_mapping.md` (스타일-용도 매핑 규칙)  
  - `design_guidelines.md` (브랜드 시각 언어 정책)  
- **/public**:  
  - `index.html` (기본 HTML 템플릿)  
  - `manifest.json` (PWA 설정)  

#### **기술적 구현**  
- **동적 색상/선명도 조정**:  
  - 사용자 피드백 수집: `localStorage` 또는 서버 저장  
  - 색상 변환 알고리즘: `#F4A261` 또는 `#2A363B` 기반  
  - CSS 필터: `brightness()` 함수 활용  
- **스타일-용도 매핑 시스템**:  
  - `style_mapping.md` 규칙 기반 자동 스타일 추천  
  - 예시:  
    ```javascript
    const styleMapping = {
      "경고": "면형",
      "성공": "굵은 선형"
    };
    ```  
- **파일 다운로드**:  
  - 결과 후보: `canvas` 또는 `svg` 라이브러리로 생성  
  - 고해상도 파일: `300dpi` 설정으로 PDF/SVG 제공  
  - 저장 경로: `/user_data/[user_id]/[timestamp]_output.svg`  

#### **자동화 로직**  
- **사용자 피드백 기반 색상 최적화**:  
  - API 연동: 사용자 피드백 데이터 서버 전송  
  - 색상 변환: `localStorage` 기반 동적 조정

### **4. 서비스 소개 및 마케팅 문구**  
#### **헤드라인**  
*"단어 하나로, 세계를 그려내는 예술"*  

#### **본문**  
"아이디어가 시각으로 탄생하는 순간, 창의성의 경계가 허물립니다. PICTO LAB은 단순한 단어 입력만으로도, 당신의 상상력을 이끌어내는 **픽토그램 생성 기술**을 제공합니다.  
- **직관적인 디자인**: 복잡한 설정 없이, 단어 하나로 즉각적인 결과를 경험하세요.  
- **감성적 표현**: 선형, 면형, 굵은 선형 등 다양한 스타일로, 당신의 감정과 메시지를 정교하게 전달하세요.  
- **전문가의 정교함**: 고해상도 PNG/SVG 파일로, 프로젝트의 완성도를 한층 업그레이드하세요."  

#### **UX 카피**  
- **단어 입력 시**:  
  "아이디어를 그려보세요. 단어 하나로, PICTO LAB이 당신의 상상력을 시각화합니다.  
  *이미지 생성 → 스타일 선택 → 결과 후보 → 다운로드*  
  간단한 과정으로, 당신의 감성을 표현해보세요."  
- **스타일 선택 시**:  
  "어떤 스타일이 당신의 감정을 가장 잘 표현하나요?  
  - **선형**: 정제된 감성, 전문적인 표현  
  - **면형**: 따뜻한 감정, 시각적 충만함  
  - **굵은 선형**: 강렬한 메시지, 강렬한 인상  
  당신의 감정을 선택하세요. PICTO LAB은 당신의 선택을 그대로 해석합니다."  
- **다운로드 시**:  
  "완성된 작품을 당신의 세계로 가져가세요.  
  PNG/SVG 파일로, 프로젝트에 바로 활용하세요.  
  *‘이 스타일이 마음에 드나요?’*  
  피드백은 다음의 창의성을 더해줍니다."  

#### **제안서 문구**  
- **학생/교육 기관용**:  
  "창의력을 위한 최적의 도구. PICTO LAB은 단어 입력만으로도, 학생들이 시각적으로 아이디어를 표현할 수 있는 **혁신적인 플랫폼**입니다.  
  - **저비용 고효율**: 학생이라면 무료 체험 기회를 통해 창의성을 발휘하세요.  
  - **교수의 고민 해소**: 시각적 자료를 쉽게 제작해, 수업 준비 시간을 절약하세요."  
- **기업/프로페셔널용**:  
  "업무의 효율성과 감성의 조화. PICTO LAB은 단어 입력만으로도, 기업이 필요로 하는 **시각적 커뮤니케이션 도구**입니다.
