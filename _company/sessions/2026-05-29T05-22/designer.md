### 🎨 **스케치업 이미지 보정 시각화 가이드**  
**색감 통일**과 **도판 톤앤매너** 규칙을 시각화한 예시:  

#### 1. **색감 통일 (Color Harmony)**  
- **주요 색상**:  
  - `#2A363B` (시멘트 톤, 기본 색상)  
  - `#F4A261` (동적 강조 색상, 사용자 선호 기반 동적 변환)  
  - `#000000` (검정, 배경/선명도 조절용)  
- **혼합 규칙**:  
  - `#2A363B` + `#F4A261` = **경고/성공** 시각적 구분  
  - `#2A363B` + `#000000` = **건축/도면** 시각적 구분  

#### 2. **도판 톤앤매너 (Planar Tone & Manner)**  
- **선형/면형/굵은 선형 표현**:  
  - **선형**: `#2A363B` 색상, 2px 선명도, 모서리 둥글게  
  - **면형**: `#F4A261` 색상, 100% 채우기, 테두리 1px  
  - **굵은 선형**: `#000000` 색상, 3px 선명도, 모서리 직선  
- **배치 규칙**:  
  - **스타일 선택 영역**: 좌측 (선형/면형/굵은 선형)  
  - **결과 후보 영역**: 우측 (색상 동적 변환, 툴팁: "색상은 사용자 선호 데이터 기반으로 동적으로 변환됩니다")

### 📌 **사용자 복사 가능한 프롬프트 문장**  
```text
[SketchUp 이미지 보정 프롬프트]  
1. 색상: #2A363B(기본), #F4A261(동적 강조), #000000(선명도 조절) 사용  
2. 선형/면형/굵은 선형: 선형(2px, 둥근 모서리), 면형(100% 채우기), 굵은 선형(3px, 직선 모서리)  
3. 배치: 스타일 선택 영역 좌측, 결과 후보 영역 우측  
4. 툴팁 추가: "색상은 사용자 선호 데이터 기반으로 동적으로 변환됩니다"  
```

### ✅ **체크리스트 (Checklist)**  
- [ ] 주색 `#2A363B`과 강조색 `#F4A261`을 통일적으로 사용  
- [ ] 선명도 조절 시 `#000000`으로 배경/선명도 설정  
- [ ] 선형/면형/굵은 선형 표현 규칙 확인 (2px/100%/3px)  
- [ ] 좌측 스타일 선택 영역, 우측 결과 후보 영역 배치  
- [ ] 툴팁에 "색상 동적 변환" 메시지 추가  
- [ ] 용도-스타일 매핑 표시 (예: "경고" → 면형, "성공" → 굵은 선형)
