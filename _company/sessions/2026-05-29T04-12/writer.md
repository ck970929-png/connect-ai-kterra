### 📌 1. 브랜드 톤 & 색상 사용법  
**"단순함이 진실입니다"**  
- **주 색상**: `#2A363B` (차분한 어두운 파랑) – UI 배경 및 텍스트  
- **강조 색상**: `#F4A261` (따뜻한 노란색) – 버튼/알림/강조 요소  
- **중성 색상**: `#FFFFFF` (흰색), `#E0E0E0` (회색) – 배경/분할선  

**💡 사용 팁**:  
- 텍스트는 `#2A363B`로 통일, 대비도 4.5:1 이상 유지  
- `#F4A261`은 사용자 선호 데이터 기반으로 동적 스타일 추천 시 강조 색상으로 사용  

---

### 📌 2. UI 레이아웃 패턴  
**"간결함이 창의성입니다"**  
- **탑 레이아웃**:  
  1. 검색창(단어 입력) →  
  2. 추천 결과 후보 →  
  3. 스타일 선택 패널 →  
  4. 다운로드 버튼  
- **아이콘 배치**:  
  - 선형/면형/굵은 선형 스타일은 좌측에 위치, 결과 후보는 오른쪽에 배치  

**💡 사용 팁**:  
- 요소 간 간격은 `16px` 이상 유지, 내부 요소 간 간격은 `8px`로 통일  

---

### 📌 3. 픽토그램 스타일 시스템  
**"선이 말을 대신합니다"**  
- **선형 규칙**:  
  - `stroke-width`: `2px` (기본), `3px` (강조), `1px` (미세)  
  - 모서리 곡률: `radius: 2px` (단순), `radius: 4px` (부드러움)  
- **색상 매핑**:  
  - 사용자 선호 데이터에 따라 `#F4A261` 또는 `#2A363B`로 동적 변환  

**💡 사용 팁**:  
- SVG/PNG 다운로드 시, 픽토그램은 `1:1 비율`로 출력, 배경은 투명도 `0.5`로 처리  

---

### 📌 4. 사용자 데이터 기반 추천 흐름  
**"단어 하나로, 아이콘의 세계가 열립니다"**  
1. **단어 입력**: 사용자는 검색창에서 단어를 입력합니다.  
2. **동적 스타일 추천**: 시스템은 사용자 선호 데이터를 기반으로 선형/면형/굵은 선형 스타일을 제안합니다.  
3. **스타일 선택**: 사용자는 추천된 스타일 중 하나를 선택합니다.  
4. **다운로드**: 선택된 아이콘은 SVG/PNG로 다운로드됩니다.  

**💡 핵심**:  
- `#F4A261`은 강조 색상으로, 사용자 선호도가 높은 스타일에 자동 적용
