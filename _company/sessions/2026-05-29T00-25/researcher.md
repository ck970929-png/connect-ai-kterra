### 🎯 **PICTO LAB 스타일 유사도 계산 로직 설계 (Researcher)**  
#### **1. 데이터 기반 선호 패턴 분석**  
- **데이터 출처**: `data/preferences.json` (최근 3개월 사용자 선택 로그)  
- **분석 항목**:  
  - **단어-스타일 매핑**: 특정 단어(예: "도서관", "정원")에 대한 선호 스타일(선형, 면형, 굵은 선형) 빈도 통계  
  - **시각적 요소 상관관계**: 선형/면형/굵은 선형의 특성(선두께, 채도, 곡률)을 기반으로 유사도 분석  
  - **시간 기반 가중치**: 최근 24시간 내 선택 빈도를 기준으로 스타일 우선순위 조정  

#### **2. 유사도 계산 로직 설계**  
- **스타일 특성 정의**:  
  - **선형 (Linear)**: 선두께 `1px`, 채도 `0%`, 곡률 `0%`  
  - **면형 (Planar)**: 선두께 `0px`, 채도 `100%`, 곡률 `50%`  
  - **굵은 선형 (Thick Linear)**: 선두께 `3px`, 채도 `0%`, 곡률 `0%`  
- **유사도 계산 방식**:  
  - **특성 유사도**: 두 스타일 간 선두께/채도/곡률의 차이를 기반으로 가중 평균 계산  
    - 예: 선형 vs 굵은 선형 → 선두께 차이(2px) × 가중치(0.5) + 채도 차이(0) + 곡률 차이(0) = 1.0  
  - **사용자 선호도 가중치**: 최근 3개월 내 해당 스타일의 선택 빈도를 기준으로 유사도 조정  

#### **3. 시스템 통합 방안**  
- **코드 연동**:  
  - `src/utils/api.js`의 `dynamicStyleRecommendation` 함수에 유사도 계산 로직 삽입  
  - 사용자 입력 단어에 해당하는 스타일 목록을 필터링 후, 유사도 기반 추천  
- **데이터 구조**:  
  - `data/style_similarity.json` 생성: 스타일 간 유사도 테이블 저장  
    ```json
    {
      "linear": {"planar": 0.3, "thick_linear": 0.8},
      "planar": {"linear": 0.3, "thick_linear": 0.5},
      "thick_linear": {"linear": 0.8, "planar": 0.5}
    }
    ```  
- **예외 처리**:  
  - 사용자 선택 데이터 없을 시, `data/style_similarity.json` 기준으로 기본 유사도 적용

     }
